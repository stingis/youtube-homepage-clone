This is a static clone of the YouTube homepage, built with HTML and CSS only, focusing on replicating its interface design and responsiveness across screen sizes, including the header, sidebar, and video layout. It features elements such as Flexbox, Grid, hover effects, and media queries, without any scripting involved.
